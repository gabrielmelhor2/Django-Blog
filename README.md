# Django Blog

Blog Django Treinamento é o back-end de um aplicativo FullStack para publicação de posts. Desenvolvido em Python 3.12.1 com o framework Django 5.2.8, implementa a busca e exibição de posts, seguindo boas práticas de segurança e estruturação.

## Funcionalidades
- Busca por título ou conteúdo
- Listagem de posts
- Interface web simples

## Competências
Python · Django · SQLite · Templates HTML

## Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/gabrielmelhor2/Django-Blog.git
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute as migrações:
   ```bash
   python manage.py migrate
   ```
4. Inicie o servidor:
   ```bash
   python manage.py runserver
   ```
